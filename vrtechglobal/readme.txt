VRTech Global Theme

Installation
1) Zip the folder `vrtechglobal` and upload via Appearance > Themes > Add New > Upload, or copy the folder to wp-content/themes/.
2) Activate the theme. It will create demo pages (Home, About, Services, Blog, Careers, Contact) and a primary menu.
3) Set logo at Appearance > Customize > Site Identity.
4) Optional: Customize hero slides at Appearance > Customize > Homepage. Provide JSON array of slides.
5) Contact form will send to admin email. Override at Appearance > Customize > Site Identity > Contact form recipient email.

Features
- Responsive design, dark modern look
- Animated sections (AOS)
- Simple hero slider (JS)
- Services grid, Industries chips, Applications cards
- Contact form with nonce and email delivery (uses wp_mail)
- Case Studies CPT (optional)
